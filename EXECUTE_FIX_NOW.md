# Выполните исправления на сервере

Поскольку автоматическое изменение workflow файлов ограничено токеном, выполните следующую команду на сервере:

```bash
curl -sSL https://raw.githubusercontent.com/buranovt2025-jpg/gogotest/main/server/deploy/auto-fix.sh | bash
```

Этот скрипт:
1. Клонирует/обновит репозиторий
2. Пересоберет API с исправлениями циклической зависимости
3. Скопирует файлы в рабочую директорию
4. Переустановит зависимости
5. Перезапустит API через PM2
6. Проверит работоспособность

После выполнения проверьте:
```bash
curl http://134.122.77.41/api/health
```

И откройте http://134.122.77.41/login
